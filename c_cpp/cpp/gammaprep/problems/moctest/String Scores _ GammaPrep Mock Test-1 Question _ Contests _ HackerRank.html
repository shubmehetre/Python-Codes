      String Scores

      • Problem
      • Submissions
      • Leaderboard

      Given a string S of length N which can consist of lowercase letters, uppercase letters and digits [0-9]. You are supposed to find the score of string.

      Score of string is initially 0 and later defined as:

      For each character ‘ch’ of string, if ‘ch’ is:

      • A lowercase consonant, then score = score + 1
      • A lowercase vowel, then score = score + 2
      • An uppercase consonant, then score = score – 1
      • An uppercase vowel, then score = score – 2
      • A digit, then score = score + (numerical value of digit)

      If final score is negative then score is normalized to 0.

      Given a string find its score.

      Example:

      For N = 5, S = “abC12” then Score = ‘a’ + ‘b’ + ‘C’ + ‘1’ + ‘2’ = 2 + 1 – 1 + 1 + 2 = 5

      For N = 7, S = “AbaB45a”, then Score = ‘A’ + ‘b’ + ‘a’ + ‘B’ + ‘4’ + ‘5’ + ‘a’ = -2 + 1 + 2 – 1 + 4 + 5 + 2 = 11

      Input Format

      First line contains an integer T – No. of test cases

      Then T test cases follow:

      • First Line contains an integer N – length of string
      • Second Line contains String S

      Constraints

      1<=T<=1000

      1 <= N <= 105

      Output Format

      For each test case, print the score of string in separate line.

      Sample Input 0

      3
5
abC12
7
AbaB45a
3
Aaa

      Sample Output 0

      5
11
